## Quickstart

1. Clone repository
   ``` shell
   git clone https://github.com/glichtner/ceosys.git
   ```
2. Install dependencies
   ``` shell
   pip install connexion[swagger-ui]
   ```
3. Run app.py
   ``` shell
   cd ceosys
   python app.py
   ```
4. Visit http://localhost:8088/ui
